End-effector of robot arm is a 2-finger parallel jaw gripper designed for maintenance
panel operation (REQ-GEN-090) and probe collection (REQ-GEN-100).

There is a unified easy-to-switch connection between end-effector and arm,
giving us the ability to switch end-effectors to be optimized for specific tasks.

% TODO: Replace [180] with actual max opening from CAD, add min opening value
The gripper mechanism provides a maximum opening width of 180\,mm and a
grip force of [TBD]\,N. The fingers employ a hybrid construction:
load cells serve as the structural base providing integrated force sensing,
CNC-machined aluminum ensures rigidity, 3D-printed components enable custom geometry,
and textured rubber contact surfaces maximize friction while
conforming to irregular object shapes.

% TODO: Add NEMA motor model number (NEMA 17/23/24?), gear reduction ratio if any
Actuation is provided by a high-torque NEMA motor
delivering [TBD]\,Nm torque. The integrated load cells enable closed-loop
force control, essential for applying calibrated pressure to maintenance panel
switches without over- or under-pressing.

% TODO: Verify 6-DOF count matches actual design in manipulator kinematics section
6-DOF arm kinematics (\autoref{par:manipulator_configuration_kinematics}) allows the gripper to approach
maintenance panel elements (switches, buttons, electromagnets, plugs)
from arbitrary orientations as required for human-ergonomic interfaces
(REQ-GEN-090, REQ-DES-140). A camera mounted adjacent to the gripper provides
visual feedback for operator and computer vision algorithm during maintenance
and probe collecting tasks.
